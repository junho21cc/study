DFS(Depth-Fist-Search): Stack으로 구현

Tree에서 inoder, preorder, postorder 도 DFS라고 한다.
child node에 child node 순회, null값까지

순회 방법
1. 자기 자신을 스택에 넣는다.
2. 자기 자신은 스택에서 뺀다.
3. 자기 자신의 인접노드를 스택에 넣는다.
4. 자기 자신을 출력한다.
5. 2번부터 실행한다.

1$ 절대로 중복해서 스택에 넣지 않는다.
2$ 인접노드가 없으면 그냥 꺼낸다.
3$ 인접노드가 스택에 넣은 상태라고 표시되어있다면 넣지 않는다.



BFS(Breadth-First-Search): Qeue로 구현

자신의 child node들을 모두 순회 그다음 child node 순회, null값까지

순회 방법
1. 자기 자신을 큐에 넣는다.
2. 자기 자신은 큐에서 뺀다.
3. 자기 자신의 인접노드를 큐에 넣는다.
4. 자기 자신을 출력한다.
5. 2번부터 실행한다.

1$ 절대로 중복해서 큐에 넣지 않는다.
2$ 인접노드가 없으면 그냥 꺼낸다.
3$ 인접노드가 큐에 넣은 상태라고 표시되어있다면 넣지 않는다.
